2016大总结,2017大畅想...

发表于 2017-02-08 | 分类于 生活志

前言

今年的总结是有史以来最晚的一次，16年忙忙碌碌，兢兢战战到了3月份才想起来忘记了年度的总结，然后拖延症又犯了一直到快年中了才发布现在的这份总结报告（发布时日期一[hou]不[yan]小[wu]心[chi]的标记为了春节当天）。当然以后的报告一定在春节左右发。根据往年的规范，现在总结一下15年的畅想把。

回顾

(未完成，评分2) 每周一本书 ：去年确实看了不少书，但是远远没有达到每周一本的目标，其实我个人总结我还是比较擅长看工具书，或者把书当工具来看，今年我决定再尝试一把，为自己订一份书单

(未完成，评分1) 参与各项运动 : 不说了，说多了都是泪，就踢了几次球

(未完成，评分0) MBA : 没开始

(进行中，评分3) 打造一款开源软件 : 今年是尝试的一年，没有很大的飞跃，后面详细说一下。

(未完成，评分1) 计划经济 : 未实施，计划经历是个很耗费经历的事情，尝试了1个月，我和媳妇都不擅长。

(未完成，评分0) Baby : 没

(已完成，评分10) 买房 : 去年买了房子，总算见识了买房不像买白菜一样，买家，卖家，中介，担保各种斗智斗勇让这个去年占了我一大半的经历！

(未完成，评分0) 个人项目收入项目翻番 : 个人项目中断了

2016大总结

Life

读书篇
其实这个和职业更贴近，我读的书里面绝大部分是职业相关的，下面就是今年度读的书了，不是很全也差不多。

日常篇
今年的生活上还是挺丰富多彩的，每年都会跟大学老友聚集次，和大龙哥的张北草原之行给扣了6分，骚气的二妹和一顿被打博博也结婚了，只可惜没有参加博博的婚礼，凯哥和轩轩就差你们了！







滑雪篇
今年滑雪终于又上了一个台阶，并且开始玩单板，我发现单板确实要比双板简单，而且快感不减，目前已经可以上高级道了，明年要去崇礼滑几次！






购房篇
今年的生活上我着重写一下买房经历，真是痛并快乐着，而且也告诫要买二手房的小朋友们，买房真真的不是一个简单的事情。
今年大约从9月份就开始看房子了，北京房价5年都没有大涨过，但是就在15年年底到16年年中，房间又上了高速。整体房价接近翻番，也正因如此才下定决心一定买房。9月份开始看房子，因为我还是不太想要一个小的过渡房，起初都是在北七家看，我的买房观点是要买一个有特点的房子，比如一层带院子或者顶层可做阁楼，或者其他的带个露台都可以，这种房子可以给自己预留更多的想象空间而且我感觉对后期的升值还有又很大影响的。跟无数中介联系，电话天天都被打爆了，基本上可以说让他们给带入门了，比如交易流程啊，贷款流程啊，高低过之类的，好多名词在多个中介中都有不同的解释，总结来看就是为了卖房不择手段，管你真的假的，签了合同交了中介非就行了！前期听各路中介的信息都是雾里看花根本看不清本质，最终我们把看房的重点从北七家移至回龙观，这里是程序员的聚集之地，而且距离西二旗开车很近，最后终于定了一套一层带花园的房子，来看看完整的时间线：

整个购房周期横跨6个月，中间两次想解约，差点跟卖家恼了，房子至今在装修，还没有入住，今年生活上的一个重点事情就是房子装修，装修还真的非功夫，为这个我连AutoCAD都给用上了，最后发现原来有傻瓜话工具，囧，放张渲染图。






出游篇

今年出游了两次，一次去了菲律宾，一次又重游了马来西亚；菲律宾是公司组织的旅游，本来是由领导带队，但临近出发时临到临时有事就变成我带队了，菲律宾日常沟通用英语还沟通得来，这次菲律宾游是我感觉最畅快的一次，我个人比较喜欢更接近原生态的地方，菲律宾我们去后租了一个摩托车，穿山越岭发现了一个没有人的悬崖边，螃蟹、各种各样的鱼、海胆遍地都是；而且菲律宾的海真的比马来西亚要好很多，我们自己买材料编网，租一个鱼人的小船浮浅真的太给力了，不知道何时能再去这个小岛，也不知下次再去时是否已被开发。

今年过年带老爸老妈和媳妇一块去了马来西亚兰卡威过年，这也是老爸老妈第一次出国，整体行程和过程都还不错。






Career

今年开始全面覆盖运维研发及运维的全部技术线，摘一段我的职业简介

负责目前系统管理部下4个团队30+人的技术管理工作
团队包括:系统团队(SA),网站运维团队(SRE),项目管理团队(PMO),运维研发团队(DEVOPS)
成员包括：项目经理,技术经理,运维研发工程师,BI工程师,需求分析师,SRE运维工程师,Linux系统工程师,网络工程师
业务范围：
混合云业务(SA):光宇私有云的建设及维护工作，涉及7个IDC机房，200+网络设备，1000+实体机物理设备，3000+虚拟化设备；阿里，腾讯，金山，Ucloud等多家共有云平台的应用运维；同时包含公司内网维护工作。
网站平台业务(SRE):光宇账号,充值,道具交易平台,手游SDK(gyyx.cn),飞猪游戏(wyx.cn),集团等网站平台业务的运维工作,其中gyyx.cn日均PV1.5亿
运维研发(DEVOPS):负责运维自动化平台，运维监控平台，大数据平台，运维管理平台的研发工作，其中自动化运维平台以自研的自动化开发平台Elves为基础管理超过2000台以上设备;运维监控平台雷达系统全面覆盖所有内部设备;大数据平台依托Hadoop/Hive建立数据仓库，目前整体容量在50TB以上;运维管理平台如故障管理系统，运维工单系统为目前日常运维提供规范及便捷服务
项目管理(PMO):全面覆盖目前部门内项目(运维产品)的管理工作，制定了部门敏捷开发/运维规范，目前全团队均按照此规范进行日常的项目/迭代工作中。
*个人业绩
管理上制定部门结构，实现部门团队的融合与再分配，并建立运维及研发梯队;业务上把控并制定项目管理流程，制定网站运维规范，混合云运维规范;技术上参与业务线研发的架构制定，自动化开发平台Elves作者，建立并推进php,java,python运维研发技术体系

这已经是我在光宇呆的第五个年头了，今年在职业思想上有波动，差点去了百度外卖，但是最终还是选择留下了，自己在职场上的思想也在不断的进步，站的越高越能看清一个职业的路线与发展，自己在运维管理及技术上的思路又有了更全年的提升，这也是我选择留下的原因，当你只是一个工程师的时候你的视野可能无法比较全面的认知一整套体系，站在高点俯视才能一栏全局。






一年中大大小小的事情太多，附述职报告的一个回顾章节
