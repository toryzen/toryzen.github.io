2018大总结,2019大畅想...

发表于 2019-02-15 | 分类于 生活志

前言

不知不觉总结已经写了9年，而且人也马上奔3了，日子感觉越过越快，一眨眼的功夫1年过去了，为留下自己每年的轨迹总结还得继续。去年抢注了我本博客域名的那畜生竟然还又给续费了，本来想到期转回来了，再等1年，我看他能续费几年。

2018大总结

生活志

整理自己这一年的生活，发现轨迹越来越模式化，基本上就是家-公司，公司-家，生活半径5公里，以往需要出门办的事也几乎都可以网上解决，也就偶尔商场看个电影，逛逛宜家，滑个雪，工作也基本上不出差，还真没有别的什么了，其他时间就宅在家里，没事刷刷美团看看有没有什么新鲜的娱乐（但还真没发现啥）。全年空闲时外出时间不超过10%，当然这也是有原因的，上半年老婆怀孕，得宅在家里，下半年儿子出生，得宅在家里。翻了一遍手机的照片，讲讲下面几个事情吧。

家装齐活了

今年本着自己动手一点点来的原则，总算把房子里里外外的装完了。每个周末留点活，屋内自己装个水台、装个热水器，画个画（酱油画，都不好意思挂墙上），屋外动手做了一个2米长的花架等等等，家装工具装了满满的一个壁橱，原先知道的不知道到叫上名的交不上名的工具先买来用上再说。再有后院烧烤没少吃，天气不错就吃烧烤，省时省力，明年继续。






两豆与两小豆儿

今年家里来了三个小过客，一只兔子，老妈心心念念的要来一只兔子，天天被两只傻狗吓的魂都没了，人还得天天出去拔草，后来这草实在是不好拔，最终给送人了。还有两个过客是黑豆红豆的宝宝，别看他们年纪不大，这么快就当爸妈了，而且是1:1克隆，小黑豆和小红豆，跟他们爸妈简直是一个模子刻出来的。红豆生宝宝之前看网上说二哈不会自己生，需要给接生，都准备好了，某天回家后都已经给生完叼窝里去了，生物的本能还是很强大的。






赔钱了赔钱了

在墙外天天高喊人民币要崩盘的呼声中，去银行换了美元现金放保险柜，买了一大堆外汇，然….央大妈一出手，人民币呼呼的升值，外汇全部爆仓，可怜的账户只剩下个位数美元，保险柜也没那么”沉”了，真实郁闷啊。说来我还真没有投资的命，当年买股票也是，买了呼呼的跌，卖了呼呼的涨，有啥办法给破一下么。(不用怀疑，我就在那个最高点入的场)






我家滴小宝宝呦

铛!铛!铛!铛! 今年喜当爹了，家中填新丁。老婆费劲九牛二虎之力终于给卸货了，急诊都跑了4-5趟，最后连帐篷我都支到产房外面了，准备打持久战，一觉还没睡完，护士找来说生啦。6斤整。这刚出生可是真够难看的哈。







随后这起名又成了一大困难事，原先我家这啥也不信随便起的，后来心血来潮找姑父（已经化身大师了）算了一下，五行不全名字补吧，好不容易起了一个，想想这八字都算了，再加个数理吧，可谓一波三折截止到报出生证明最后时刻终于定下来了，期间起过：黄牧尧（最后”牧尧”留作小名了）、黄玙璠、黄彦云。最终定名：黄泽皓，这回五行全了，数理也都是大吉了！





如今宝宝已经半岁多了，越长越好看了，快快茁壮成长吧！








爱技术、爱生活

今年开始有幸参加了多个itechclub、麒麟会的活动，石家庄首次参与华北分会德州友谊赛靠新手运拿了个第一（后来再玩没一次赢钱的..），然后10年后的重游重庆，黑科技挑战赛、火锅等也别有一番风味！






工作纪

生活说完了说工作，经过7年的职场经历，今年终于升任部门经理，覆盖50人左右系统管理部的整体管理工作，下面简单从晋升和业务上说下。

晋升报告

先放一下没有敏感信息部分的晋升述职，从个人的理解部分：技术型管理思路、部门战略方向的PPT单页。






从这些年的部门技术、管理建设部分：技术架构体系、项目管理体系、部门管理体系的PPT单页。







业务范畴

在业务范畴上，今年主要追加了两块内容，游戏运维的管理与IT的管理，实现了部门内全部业务上的管理。

  • 负责公司游戏运维组织与管理工作, 保障游戏的稳定运营, 快速的故障响应处理, 优化游戏架构, 提升游戏的整体可靠性与稳定性, 减少游戏故障率
  • 负责公司网站类项目对外业务平台运维组织与管理工作, 保障社区的稳定运营, 快速的故障响应处理, 优化社区架构, 提升社区的整体可靠性与稳定性, 减少社区故障率。
  • 负责公司网络平台（IDC网络, 公司内网）的运维组织与管理工作, 规范并优化体系建设, 提升网络可靠性与稳定性, 减少网络相关故障。
  • 负责公司虚拟化平台（KVM, 公有云）的运维组织与管理工作, 规范并优化体系建设, 提升系统可靠性与稳定性, 减少系统相关故障
  • 负责公司运维支撑平台、大数据平台的建设、开发与维护工作, 负责公司运营数据分析平台的建设、开发与维护工作
  • 负责公司办公电脑软硬件维护、办公网络、电话、打印机设备、IDC机房、设备库房等管理工作。

目前团队的整体实力、业务的稳定性均达到一定高度，希望明年可以与团队一起进行更高难度的挑战。

2019大畅想

重看一遍18年的大畅想，简直惨不忍睹…，算了那就不看了，直接想想19年吧！

  • 兴趣，为自己发掘一个新的兴趣爱好，个人兴趣爱好太少了！
  • 阅读，读20本以上的书，让自己多一些文化气息。
  • 项目，争取年内推出一个盈利性质创新项目。
  • 眼镜，不想再戴眼镜了，研究研究做个手术。
  • 户口，争取今年解决一下户口问题
  • 预留，…..

希望在新的一年里，生活上顺顺利利，职场上有新的挑战！
