2017大总结,2018大畅想...

发表于 2018-02-11 | 分类于 生活志

前言

鉴于去年的总结拖延症问题，今年的总结整理的时间还是比较早的，回顾今年的一整年的生活及工作，各方面都在有条不紊的进步，但整体来说我自我感觉对本年度并不是特别满意，其实这也是一个矛盾的体现，生活、工作上事事顺利是努力的目标，但向这个目标努力也越发感到危机，今年比较郁闷的是原先的Blog域名忘记续费了，然后被抢注了，还挂上了六*彩，郁闷！等它到期再买回来吧。好了，感慨就发到这了，下面开始回顾2017年度的大事件吧！

2017大总结

比较惭愧，“2016大总结，2017大畅想”中没有畅想！今年好好畅想。

生活篇

阅读

今年没有读几本书，自己的阅读能力这几年在逐年下降，现在越来越不愿意进行完整化的阅读了，但碎片化的阅读又非常没有营养，今年要立志改善！尽量避免碎片化阅读，像什么刷今日头条、脉脉、微博等今年要一刀切拒绝！阅读的目标转向得到、即刻、知乎、豆瓣。并且今年重启豆瓣主页，在阅读的书籍方面，仍然以技术方向为主（偏技术底层），管理、民科为辅的阅读。

装修

今年在装修上可花了大力气，很多地方都是自己亲自上阵，解锁了众多的技能，像瓷砖填缝、灯具安装、橱子柜子制作、重整围栏等一票的众多事情都是自己亲自上手做的，很多的事情上不仅仅是消耗体力、而且还消耗脑力，像灯具的安装重新搞了一下电路、厨子柜子制作从草图大师开始到买板材、开板材、组装等一系列的操作，从最开始的一个小电视柜用了1天，到最后一整面墙的柜子（4米长*2.4米高）用了1天，从自己琢磨怎么组装到最后参考宜家的组装方式确实学到了不少知识，就是感觉开版还是太危险了，个人还是比较抵触的，可能后期这块技能应该发挥余地会比较小，个人还是比较倾向于无安全隐患的事情。






红豆和黑豆

经过了N多次的思想斗争，最终今年请了两个宝进家，红豆和黑豆，来的时候3个月不到，3、4斤的样子，经过4-5个月的猪式喂养，两个小家伙很快就变成了两个大家伙，每个40斤！现在两个家伙壮成牛，跑起来像两个肉球撞来撞去，拉车都不成问题，出门被拽着各种乱跑，估计以后就降不住了啊！

钢琴

今年买了架电钢琴，找了很久也没找到比较满意的，目标是可以插手机连接的方便电钢琴，但又想要钢琴的样子当个装饰，最后选择了一个，只能说还凑合吧，截至目前2个月差不多能弹两个曲子了，小步舞曲和简易卡农，明年要多学几个谱子。

滑雪

今年的滑雪进阶非常大，崇礼的高级道全部刷了一遍，快感足足的！

骑行

其实这个章节去年就应该体现，去年入手了一辆摩托车，当然时乞丐版，跑去洛阳学的摩托车驾证，听着摩托车的轰鸣声、疾风吹过耳边的感觉很有Feel..18年计划升级一款。

全民K歌 && 抖音

今年开始玩了玩全民K歌和抖音，唱了10首单曲、录了15个短视频，挺有意思的。






娃

今年是个扎堆生娃的一年，有了大龙哥的可乐打头阵、二妹、烤肠都已上岸，凯哥的求婚也超级成功，明年”黄豆”也来啦！

职场篇

今年在职场上走的比较顺利，整体的业务范围没有较大范围的变化，从主管升为副经理，团队的实力在稳步提升。

GITC

今年度有幸作为GITC讲师，进行了一场运维自动化相关主题的演讲工作，效果自我感觉还是不错的，后期继续多参加此类会议。

2018大畅想

生活篇

  • 钢琴曲：今年后半年才买的琴，算是搞定了两首钢琴曲（小步舞曲，卡农简易版），18年计划至少完成3首钢琴曲，其中一首已经订好了-亡灵序曲
  • 健康的是生活与形体塑形：今年有计划做一下近视治疗手术，不想再带眼镜了，除此之外通过健身解决一下自己的驼背毛病，这一点感觉自己已经很注意了，但是从一些照片之类的地方来看还是做的不够。
  • 机械：18年再解锁一个技能，机械类，准备拆解一下摩托车，了解一下它整个的组装架构和工作原理

技能篇

  • 高数：18年准备重学一下高数，别问我为什么，就是要体验这种动脑的过程。
  • 算法与数据结构：这两部分也要重新学习一下，总感觉不扎实，但做技术确实方方面面都要用到他们

工作篇

18年的工作计划还是比较饱满的，已经初步制订了4个季度的工作计划，明年个人比较关注的两个项目。

  • 智能网关系统：基于Linux策略路由，iptables，ipset，并基于OA的人员组织结构数据、网关抓包个人带宽占用情况，结合大数据算法，基于单部门、单人进行多线路的智能切换，最大限度的利用多线路带宽。
    智能网关支持：限速：线路级限速、部门级限速、个人级限速，安全：个人上网绑定、异常机器检测， WIFI上网：无线认证系统，统计：线路带宽统计、部门带宽统计、个人带宽统计，操作：多线路智能切换、人工切换。

  • 资源成本决策系统：基于目前自有资产、机房、机架的租赁、折旧等计算模式（设置虚拟机价格体系），结合公有云的计费API，建设自动化的资源统计系统，可细节到部门级、项目级，支持已付成本的计算、成本预测。（Stream信息准确性，提供带宽接口，中心应用信息准确性，提供机房、机器信息）

其他细项

  • 翻译并维护一份开源的技术项目文档
  • 持续维护更新ELVES：SpringBoot重构ELVES核心组件
  • 持续维护更新SmartPing：集成MTR、分组等功能

希望明年可以与大家见面！
